# 💸 Bill Splitter

A free, smart bill-splitting tool — split bills equally, by base + add-ons, or fully custom. Tip is calculated and distributed automatically.

🔗 **Live Demo:** [your-url.vercel.app](https://your-url.vercel.app) ← replace with your Vercel URL

---

## Features

- ⚖️ **Equal Split** — divide the bill equally among everyone
- ➕ **Base + Add-ons** — share a common base equally, add personal extras per person
- ✏️ **Custom Split** — each person enters their own food amount manually
- ✨ **Auto Tip** — tip is split equally and added on top automatically
- 📱 **Mobile Friendly** — works on all screen sizes
- 🆓 **Free Forever** — no ads, no sign-up, no data stored anywhere

---

## Tech Stack

- [Next.js 14](https://nextjs.org/) — React framework
- [Tailwind CSS v4](https://tailwindcss.com/) — styling
- Deployed on [Vercel](https://vercel.com/) — free hobby plan

---

## Run Locally

```bash
git clone https://github.com/pravijain789/bill-splitter.git
cd bill-splitter
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
