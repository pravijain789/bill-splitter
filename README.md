# 💸 Bill Splitter

A free, smart bill-splitting tool built with Next.js. Split bills equally, by shared base + personal add-ons, or fully custom — with automatic tip calculation and distribution.

🔗 **Live Demo:** [bill-splitter-jade-delta.vercel.app](https://bill-splitter-jade-delta.vercel.app)

---

## Features

- ⚖️ **Equal Split** — divide the total bill equally among everyone
- ➕ **Base + Add-ons** — split a shared base equally, then add personal extras per person
- ✏️ **Custom Split** — each person enters their own food amount manually
- ✨ **Auto Tip** — choose from preset tip % or enter custom; tip is split equally and added automatically
- 📱 **Mobile Friendly** — works on all screen sizes
- 🆓 **Free Forever** — no ads, no sign-up, no data stored anywhere

---

## How It Works

1. Enter the total bill amount
2. Choose a tip percentage (or skip)
3. Add the people splitting the bill
4. Choose your split mode — Equal, Base + Add-ons, or Custom
5. See everyone's share instantly

## Tech Stack

- [Next.js 14](https://nextjs.org/) — React framework
- [Tailwind CSS v4](https://tailwindcss.com/) — styling

## Run Locally

```bash
git clone https://github.com/pravijain789/bill-splitter.git
cd bill-splitter
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
